# [CODEX-05] Brief Para STITCH - Landing Page Oficial do Pessoas em Acao
**Autor:** Codex GPT-5
**Identificador:** CODEX-05
**Objetivo:** fornecer ao STITCH um briefing completo de estrutura, copy e direcao visual para gerar o design system e o design da landing page do evento Pessoas em Acao.

---

## 1. Papel do STITCH neste projeto

Voce deve criar o **design system** e o **design da landing page** do evento **Pessoas em Acao**.

O objetivo nao e criar uma pagina institucional fria. O objetivo e criar uma **landing page de conversao**, com cara oficial, energia alta, sensacao premium e forte apelo emocional.

O site deve:

- parecer oficial
- parecer confiavel
- parecer premium, mas acessivel
- converter visitantes em interessados no WhatsApp
- funcionar muito bem no desktop e no mobile

---

## 2. Regra critica sobre os dados atuais

Use os dados abaixo normalmente no layout, como se fossem os dados vigentes da pagina.

### Dados atuais para usar no design

- **Data:** 2 de maio de 2026
- **Local:** Cabo Frio, RJ
- **Capacidade:** 100 vagas
- **Formato:** presencial
- **Duracao:** aproximadamente 5 horas
- **Preco:** R$150
- **CTA principal:** WhatsApp

### Regra importante

Esses dados sao **placeholders internos do projeto**, mas **isso nao deve aparecer visualmente no layout**.

- "prototipo"

A pagina deve parecer oficial.

## 5. Direcao visual

### Paleta base

Usar como base as cores do projeto:

- `#2f2d31` fundo escuro principal
- `#b35465` rosa principal
- `#eb8599` rosa de destaque
- `#5dbdf0` azul principal
- `#4e9ede` azul de apoio
- `#f0eff2` claro para contraste

### Linguagem visual

- fundo escuro predominante
- contraste alto
- glow sutil, nao exagerado
- gradientes e profundidade
- grafismos circulares ou orbitais
- clima contemporaneo, vibrante e humano
- evitar cara de template generico

### Tipografia

O sistema tipografico deve parecer:

- forte
- moderno
- marcante
- legivel

Preferencia por:

- display expressivo para headlines
- sans serif forte para corpo e interface

### Imagens

Priorizar conceito visual de:

- pessoas reais
- networking
- conexoes
- alegria
- grupo
- troca

Evitar:

- fotos corporativas frias
- poses artificiais de banco de imagem
- visual de congresso burocratico

## 9. Instrucoes de design system para o STITCH

### Componentes que o design deve resolver

- hero com CTAs
- faixa de indicadores
- cards de pilares
- cards de publico
- secao de experiencia
- checklist de beneficios
- timeline ou agenda
- cards de palestrantes
- card principal de ingresso
- accordion de FAQ
- CTA final
- rodape

### Regras de UX

- CTA principal visivel no hero
- repeticao de CTA em pontos estrategicos
- leitura facil no mobile
- alto contraste
- uma hierarquia tipografica forte
- componentes coerentes entre si

### Evitar

- cara de evento corporativo padrao
- excesso de texto espremido
- layout sem tensao visual
- estetica generica de SaaS
- visual morno
